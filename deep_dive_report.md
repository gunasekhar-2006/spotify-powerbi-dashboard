# Spotify Dashboard Deep-Dive Analysis

## Objective
Develop an interactive Power BI dashboard to analyze Spotify music trends and audio features.

## KPIs
- Total Songs
- Average Popularity
- Average Danceability
- Average Energy
- Total Genres

## Dashboard Insights

### Genre Trends
Pop-related genres dominate Spotify popularity rankings.

### Danceability Impact
Songs with higher danceability tend to achieve better popularity scores.

### Artist Analysis
Certain artists consistently maintain strong popularity across tracks.

### Audio Features
Energy and danceability strongly influence listener engagement.

## Tools Used
- Power BI
- Python
- Pandas
